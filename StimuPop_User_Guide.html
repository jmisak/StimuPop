💡 1. Introduction

What is StimuPop?

StimuPop is a powerful yet easy-to-use application that converts Excel spreadsheet data into professional PowerPoint presentations. It is designed to save you hours of manual work by automatically handling images, text, and formatting.

📷

Image Extraction

Automatically extracts embedded images from Excel cells with full fidelity.

📄

Slide Generation

Creates individual slides for each data row with consistent formatting.

🎨

Smart Sizing

Multiple sizing modes ensure uniform image appearance across all slides.

✎

Custom Formatting

Control fonts, sizes, colors, and positioning for professional results.

🔄

Dual Orientation

Supports both portrait and landscape slide orientations.

✅

Error Handling

Gracefully handles missing images without stopping the process.

Pro Tip
StimuPop is perfect for creating product catalogs, photo albums, training materials, and any presentation where each slide follows the same format.

🚀 2. Getting Started

Installation (Portable Version)

The portable version requires no installation. Simply follow these steps:

1 Extract the ZIP file to any folder on your computer
2 Open the extracted folder
3 Double-click StimuPop.exe (or StimuPop.bat) to launch the application
4 Wait for the terminal window to display Server ready! message
5 Hold Ctrl and click the localhost link, OR copy the URL and paste it into your browser
First Launch
The first time you run StimuPop, the server may take 30-60 seconds to start. Wait for the Server ready! message before clicking the link. Subsequent launches will be faster.

System Requirements

💻 Operating System Windows 10 or later (64-bit)
💾 Disk Space ~300 MB for portable version
🧮 Memory 4 GB RAM minimum
🌐 Browser Chrome, Firefox, or Edge (latest version)

Image Options

Paste images directly into Excel cells. This is the most reliable method and ensures images travel with the file.

Enter the full path to an image file (e.g., C:\Images\photo.jpg). Ensure paths are accessible from the computer running StimuPop.

Enter a web URL to an image (e.g., https://example.com/image.png). Requires internet connection during generation.

Important
Images embedded directly in Excel cells provide the most consistent results. File paths must be accessible from the computer running StimuPop.

🛠 5. Configuration Options

Basic Settings

Setting Description Default
Image Column Letter (A-Z) or name of the column containing images B
Text Columns Comma-separated list of columns for text content C,D,E,F
Font Size Default font size for all text (10-32pt) 14

Advanced Settings

Click Advanced Settings to access layout, image sizing, and alignment options:

Control how images are positioned within their bounding box:

Alignment Type Options
Vertical Alignment Top, Center (default), Bottom
Horizontal Alignment Left, Center (default), Right
Tip
Use Bottom vertical alignment for variety cards where you want images anchored to the bottom of the image area.

Control how images are sized uniformly across all slides:

Size Mode Description
Fit to Box (Recommended) Images scale to fit within Max Width and Max Height while maintaining aspect ratio.
Fit Width Only Images have fixed width, height adjusts automatically based on aspect ratio.
Fit Height Only Images have fixed height, width adjusts automatically based on aspect ratio.
Stretch to Exact Size Images are forced to exact dimensions. May distort images.

These settings are only available when using Blank slide mode (not Template mode):

Setting Description Default
Max Width Maximum image width (2.0-9.0 inches) 5.5 inches
Max Height Maximum image height (2.0-7.0 inches) 4.0 inches
Image Top Position Distance from top of slide to image 0.5 inches
Text Top Position Distance from top of slide to text area 5.0 inches
Slide Orientation Portrait (tall) or Landscape (wide) Portrait

For precise control over text column placement, enable Advanced Positioning:

Auto (default)
Text flows sequentially after the previous column.
Fixed
Text is placed at an exact position on the slide, regardless of other content.

Default fixed positions (useful for variety cards):

  • Column E: 5.0 inches from top
  • Column F: 6.5 inches from top

When using Blank mode, you can customize each text column individually:

  • Font size (8-48pt)
  • Font family (Calibri, Arial, Times New Roman, etc.)
  • Text color (color picker)
  • Bold and Italic styles
Tip
Use larger, bold fonts for titles (Column C) and smaller regular fonts for descriptions (Column D) to create visual hierarchy.

🔧 6. Troubleshooting

Application won't start
  • Ensure you extracted all files from the ZIP
  • Run as Administrator (right-click and select Run as administrator)
  • Check that antivirus is not blocking the application
Browser does not open automatically
  • Wait for Server ready! message in the terminal window
  • Hold Ctrl and click the localhost link, or copy/paste the URL
  • Manually open your browser and go to: http://localhost:8501
  • Try a different browser (Chrome recommended)
'Column not found' error
  • Check that your column letter/name matches exactly
  • Column letters are case-insensitive (B = b)
  • Column names ARE case-sensitive
Images not appearing on slides
  • Ensure images are embedded in cells, not floating
  • Check file paths are correct and accessible
  • Verify image files are JPG, PNG, or GIF format
Presentation generation is slow
  • Large images take longer to process
  • Consider resizing images in Excel before upload
  • Reduce the number of slides if testing
Text formatting not applied
  • Ensure you configured formatting in Advanced Settings
  • Check that column letters in formatting match your data
Images are different sizes on slides
  • Open Advanced Settings and set Size Mode to Fit to Box
  • Adjust Max Width and Max Height to your desired dimensions
  • All images will now scale uniformly
